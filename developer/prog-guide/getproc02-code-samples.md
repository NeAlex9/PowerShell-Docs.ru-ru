# <a name="getproc02-code-samples"></a>Примеры кода GetProc02

Ниже приведены примеры кода для командлета GetProc02 образца. Это `Get-Process` командлетов, которые описаны в [Добавление параметров командной строки этого процесса входа](../cmdlet/adding-parameters-that-process-command-line-input.md). Это `Get-Process` командлет извлекает процессов на основе их имени и затем отображает сведения о процессах в командной строке.

> [!NOTE]
> Вы можете скачать C# исходный файл (getproc02.cs) для этого командлета Get-Proc, с помощью Microsoft Windows программное обеспечение Development Kit для Windows Vista и компоненты среды выполнения .NET Framework 3.0. Инструкции по загрузке см. в разделе [как установка Windows PowerShell и загрузки пакета SDK для Windows PowerShell](/powershell/developer/installing-the-windows-powershell-sdk).
>
> Скачанный исходные файлы доступны в  **\<примеры PowerShell >** каталога.
