# <a name="runspace04-code-samples"></a>Примеры кода RunSpace04

Ниже приведен пример кода для пространства выполнения, который использует [System.Management.Automation.Runspaceinvoke](/dotnet/api/System.Management.Automation.RunspaceInvoke) класса, чтобы выполнить скрипт, который создает неустранимую ошибку. Ведущее приложение отвечает за перехват ошибок и интерпретации запись об ошибке.

> [!NOTE]
> Для этого пространства выполнения, используя Windows Software Development Kit для Windows Vista и компоненты среды выполнения Microsoft .NET Framework 3.0 можно загрузить исходный файл VB.NET (Runspace04.vb). Инструкции по загрузке см. в разделе [как установка Windows PowerShell и загрузки пакета SDK для Windows PowerShell](/powershell/developer/installing-the-windows-powershell-sdk).
