# <a name="approved-verbs-for-powershell-commands"></a>Утвержденные глаголы для команд PowerShell

PowerShell использует пару существительное-имена командлетов и их производных классов Microsoft .NET Framework.
Например `Get-Command` предоставляемые PowerShell командлет используется для получения всех команд, которые зарегистрированы в PowerShell.
Команда часть имени определяет действие, выполняемое командлетом.
В именной части имени идентифицирует сущность, на котором выполняется действие.

> [!NOTE]
> PowerShell используется термин *команда* для описания слово, которое подразумевает действия, даже если это слово не обычного глагола, на английском языке.
> Например, термин *New* является допустимым именем команды PowerShell, так как он подразумевает действия, несмотря на то, что это не команды на английском языке.

## <a name="verb-naming-rules"></a>Правила именования глагол

Ниже приведены рекомендации, которые необходимо учитывать при выборе глагола для имени командлета.

* При указании команды, мы рекомендуем использовать один из предварительно определенных глаголов, предоставляемые PowerShell (псевдонимы для этих стандартных команд, включаются в следующих таблицах).
  При использовании стандартных команд, можно обеспечить их согласованность командлеты, которые можно создать, командлеты, которые предоставляются PowerShell и командлеты, которые разработаны с другими пользователями.

* Используйте стандартные команды общие области действия и использовать параметры для дальнейшего уточнения действие командлета.

* Для обеспечения согласованности между командлеты, не используйте синонима утвержденного глагола.

* Форма только каждой из команд, перечисленных в этом разделе.
  Например, используйте «Get», но не используйте «Начало» или «Возвращает».

* Стиль языка Pascal регистр для команд.
  В стиле Pascal регистр начальной буквы каждого слова написано с прописной буквы, например «ForEach».

* Не используйте следующие команды, зарезервированных или псевдонимы.
  Эти команды используются языком PowerShell, или командлетами специальных вариантов, предоставляемые PowerShell.
  - По каждому элементу (foreach)
  - Формат (f)
  - Группы (gp)
  - Сортировка (sr)
  - TEE (te)
  - Где (ч)

## <a name="similar-verbs-for-different-actions"></a>Аналогичные команды для различных действий

Следующие аналогичные команды представляют различные действия.

### <a name="new-vs-set"></a>Новый vs. Установить
`New` Команда используется для создания нового ресурса.
`Set` Команда используется для изменения существующего ресурса, при необходимости создания ресурса, если он не существует, такие как `Set-Variable` командлета.

### <a name="find-vs-search"></a>Найти vs. Найти
`Find` Команда используется для поиска объекта.
`Search` Команда используется для создания ссылки на ресурс в контейнере.

### <a name="get-vs-read"></a>Получите vs. Чтение
`Get` Команда используется для извлечения ресурса, например файл.
`Read` Команда используется для получения сведений из источника, например файл.

### <a name="invoke-vs-start"></a>Вызовите vs. Запуск
`Invoke` Команда используется для выполнения операции, которая обычно выполняется в синхронном, такими как выполнение команды.
`Start` Команда используется для запуска операции, обычно является асинхронной операцией, например запуск процесса.

### <a name="ping-vs-test"></a>Ping vs. Тест
Используйте `Test` команды.
