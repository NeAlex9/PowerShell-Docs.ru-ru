# <a name="cmdlet-development-guidelines"></a>Рекомендации по разработке командлетов

В этом разделе представлены рекомендации по разработке, которые можно использовать для создания правильного командлетов. Используя общие функциональные возможности, предоставляемые средой выполнения Windows PowerShell и эти рекомендации, можно разрабатывать надежные командлеты с минимальными усилиями и предоставить пользователю согласованный. Кроме того позволяет снизить тестовую нагрузку, так как общие функциональные возможности не требуют повторного тестирования.
