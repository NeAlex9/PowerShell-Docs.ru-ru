# <a name="text-element-for-customitem-for-customview-for-view-format"></a>Элемент Text для элемента CustomItem для элемента CustomView для элемента View (формат)

Задает текст, который добавляется к данным, отображаемого элементом управления, например метки, квадратные скобки для заключения в них данных и пробелов для отступа данные. Этот элемент используется при определении представления пользовательского элемента управления.
