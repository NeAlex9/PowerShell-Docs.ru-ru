# <a name="controls-element-for-view-format"></a>Элемент Controls для элемента View (формат)

Определяет элементы управления представления, которые могут использоваться по определенному представлению.

Элемент представления ViewDefinitions элемент (формат) конфигурации элемент (формат) (формат) элемент Controls для представления (формат)

## <a name="syntax"></a>Синтаксис

```xml
<Controls>
  <Control>...</Control>
</Controls>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах атрибуты, дочерние элементы и родительские элементы из `Controls` элемент. Этот элемент должен иметь по крайней мере один дочерний элемент. Существует неограниченное число дочерних элементов, а также важен их порядок.
