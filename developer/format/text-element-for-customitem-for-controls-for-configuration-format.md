# <a name="text-element-for-customitem-for-controls-for-configuration-format"></a>Элемент Text для элемента CustomItem для элемента Controls для элемента Configuration (формат)

Задает текст, который добавляется к данным, отображаемого элементом управления, например метки, квадратные скобки для заключения в них данных и пробелов для отступа данные. Этот элемент используется при определении общего элемента управления, который может использоваться все представления в файле форматирования.
