# <a name="hidetableheaders-element-format"></a>Элемент HideTableHeaders (формат)

Указывает, что заголовки таблицы не отображаются.

Элемент ViewDefinitions (формат) представления (формат) TableControl-элемент (формат) HideTableHeaders элемента для TableControl (формат)

## <a name="syntax"></a>Синтаксис

```vb
<HideTableHeaders/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах атрибуты, дочерние элементы и родительский элемент `HideTableHeaders` элемент.

### <a name="attributes"></a>Атрибуты

Нет.

### <a name="child-elements"></a>Дочерние элементы

Нет.

### <a name="parent-elements"></a>Родительские элементы

|Элемент|Описание|
|-------------|-----------------|
|[TableControl-элемент (формат)](./tablecontrol-element-format.md)|Определяет формат таблицы для представления.|

## <a name="text-value"></a>Текстовое значение

Укажите `true` скрытие заголовков таблицы.
