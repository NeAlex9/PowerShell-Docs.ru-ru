# <a name="text-element-for-customitem-for-controls-for-view-format"></a>Элемент Text для элемента CustomItem для элемента Controls для элемента View (формат)

Задает текст, который добавляется к данным, отображаемого элементом управления, например метки, квадратные скобки для заключения в них данных и пробелов для отступа данные. Этот элемент используется при определении элементов управления, которые могут использоваться представлением.
