# Модуль Microsoft.PowerShell.Core

## Описание

В этом разделе содержатся разделы справки по командлетам, которые устанавливаются с помощью модуля PowerShell Microsoft. PowerShell. Core. Модуль Core содержит командлеты и поставщики, которые управляют базовыми функциями PowerShell.
