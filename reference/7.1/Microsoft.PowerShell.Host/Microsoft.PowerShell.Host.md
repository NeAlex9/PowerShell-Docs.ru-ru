# Модуль Microsoft.PowerShell.Host

## Описание

В этом разделе содержатся разделы справки по командлетам, которые устанавливаются вместе с модулем PowerShell Microsoft. PowerShell. host. Модуль Host содержит командлеты, управляющие данными с помощью основных программ.

## Командлеты Microsoft.PowerShell.Host

### [Start-Transcript](Start-Transcript.md)
Создает запись для всего сеанса PowerShell или его части в текстовый файл.

### [Stop-Transcript](Stop-Transcript.md)
Останавливает запись.
