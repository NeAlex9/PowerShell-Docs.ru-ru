# Модуль Microsoft.PowerShell.Utility

## Описание

В этом разделе содержатся разделы справки по командлетам, которые устанавливаются вместе с модулем Microsoft. PowerShell. Utility. Этот модуль содержит командлеты, управляющие основными функциями PowerShell.
