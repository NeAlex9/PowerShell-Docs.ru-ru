# Модуль PSWorkflowUtility

## Описание

Этот раздел содержит разделы справки по командам, которые экспортирует модуль PSWorkflowUtility Windows PowerShell. Этот модуль содержит рабочий процесс, который поддерживает компонент рабочих процессов Windows PowerShell.

## Командлеты PSWorkflowUtility

### [Invoke-AsWorkflow](Invoke-AsWorkflow.md)
Выполняет команду или выражение как рабочий процесс Windows PowerShell.
