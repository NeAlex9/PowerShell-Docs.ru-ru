# Модуль PSWorkflow

## Описание

Этот раздел содержит разделы справки по командлетам, устанавливаемым вместе с модулем PSWorkflow Windows PowerShell, который содержит командлеты, поддерживающие компонент рабочих процессов Windows PowerShell.

## Командлеты PSWorkflow

### [New-PSWorkflowExecutionOption](New-PSWorkflowExecutionOption.md)
Создает объект, содержащий параметры конфигурации сеанса для сеансов рабочего процесса.

### [New-PSWorkflowSession](New-PSWorkflowSession.md)
Создает сеанс рабочего процесса.
