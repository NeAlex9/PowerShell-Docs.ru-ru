# Модуль Microsoft.PowerShell.ODataUtils

## Описание

В этом разделе содержатся разделы справки по модулю Microsoft.PowerShell.ODataUtils. Модуль Microsoft. PowerShell. ODataUtils устанавливается вместе с Windows PowerShell 5,1.

## Командлеты Microsoft. PowerShell. ODataUtils

### [Export-ODataEndpointProxy](Export-ODataEndpointProxy.md)
Создает модуль, содержащий командлеты для управления конечной точкой OData.
