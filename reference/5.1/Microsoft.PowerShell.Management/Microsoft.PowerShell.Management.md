# Модуль Microsoft.PowerShell.Management

## Описание

В этом разделе содержатся разделы справки по командлетам, которые устанавливаются с помощью модуля PowerShell Microsoft. PowerShell. Management. Модуль управления содержит командлеты, помогающие управлять Windows в PowerShell.

## Командлеты Microsoft. PowerShell. Management

### [Add-Computer](Add-Computer.md)
Добавляет локальный компьютер в домен или рабочую группу.

### [Add-Content](Add-Content.md)
Добавляет содержимое в указанные элементы, например слова в файл.

### [Checkpoint-Computer](Checkpoint-Computer.md)
Создает точку восстановления системы на локальном компьютере.

### [Clear-Content](Clear-Content.md)
Удаляет содержимое элемента без удаления самого элемента.

### [Clear-EventLog](Clear-EventLog.md)
Удаляет все записи из указанных журналов событий на локальном или удаленном компьютере.

### [Clear-Item](Clear-Item.md)
Удаляет содержимое элемента, но не удаляет его.

### [Clear-ItemProperty](Clear-ItemProperty.md)
Удаляет значение свойства без удаления самого свойства.

### [Clear-RecycleBin](Clear-RecycleBin.md)
Очищает содержимое корзины.

### [Complete-Transaction](Complete-Transaction.md)
Фиксирует активную транзакцию.

### [Convert-Path](Convert-Path.md)
