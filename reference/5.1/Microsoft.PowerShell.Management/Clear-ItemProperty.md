# Clear-ItemProperty

## Краткий обзор
Удаляет значение свойства без удаления самого свойства.

## SYNTAX

### Путь (по умолчанию)

```
Clear-ItemProperty [-Path] <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>] [-WhatIf] [-Confirm]
 [-UseTransaction] [<CommonParameters>]
```

### LiteralPath

```
Clear-ItemProperty -LiteralPath <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>] [-WhatIf] [-Confirm]
 [-UseTransaction] [<CommonParameters>]
```

## DESCRIPTION

`Clear-ItemProperty`Командлет очищает значение свойства, но не удаляет свойство.
Данный командлет можно использовать для удаления данных параметра реестра.

## Примеры

### Пример 1. Удаление значения раздела реестра

Эта команда очищает данные в параметре реестра "Options" в подразделе "MyApp" раздела `HKEY_LOCAL_MACHINE\Software\MyCompany` .

```powershell
Clear-ItemProperty -Path "HKLM:\Software\MyCompany\MyApp" -Name "Options"
```

### System.String

## Выходные данные

### Нет или System.Management.Automation.PSCustomObject

При использовании параметра *PassThru* `Clear-ItemProperty` создает объект **PSCustomObject** , представляющий свойство очистки элемента.
В противном случае командлет не формирует никаких выходных данных.

## ПРИМЕЧАНИЯ

Можно использовать `Clear-ItemProperty` для удаления данных из значений реестра без удаления значения. Если значение имеет тип данных Binary или DWORD, то при его очистке ему присваивается значение 0. В противном случае присваивается пустое значение.

`Clear-ItemProperty`Командлет предназначен для работы с данными, предоставляемыми любым поставщиком. Чтобы вывести список поставщиков, доступных в данном сеансе, введите командлет `Get-PSProvider`. Дополнительные сведения см. в разделе about_Providers.
