## <a name="getting-services"></a><span data-ttu-id="3dfd7-106">Получение служб</span><span class="sxs-lookup"><span data-stu-id="3dfd7-106">Getting Services</span></span>

<span data-ttu-id="3dfd7-107">Получить службы на локальном или удаленном компьютере можно с помощью командлета **Get-Service**.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-107">You can get the services on a local or remote computer by using the **Get-Service** cmdlet.</span></span> <span data-ttu-id="3dfd7-108">Как и в случае с **Get-Process**, использование команды **Get-Service** без параметров возвращает все службы.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-108">As with **Get-Process**, using the **Get-Service** command without parameters returns all services.</span></span> <span data-ttu-id="3dfd7-109">Можно фильтровать по имени, даже используя звездочку как подстановочный знак:</span><span class="sxs-lookup"><span data-stu-id="3dfd7-109">You can filter by name, even using an asterisk as a wildcard:</span></span>

------   ----               -----------
Running  seclogon           Secondary Logon
Running  SENS               System Event Notification
Stopped  ServiceLayer       ServiceLayer

<span data-ttu-id="3dfd7-110">Так как реальное имя службы не всегда очевидно, может потребоваться найти службы по отображаемому имени.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-110">Because it is not always obvious what the real name for the service is, you may find you need to find services by display name.</span></span> <span data-ttu-id="3dfd7-111">Это можно сделать с использованием определенного имени, подстановочных знаков или списка отображаемых имен:</span><span class="sxs-lookup"><span data-stu-id="3dfd7-111">You can do this by specific name, using wildcards, or using a list of display names:</span></span>

Running  SamSs              Security Accounts Manager
Running  seclogon           Secondary Logon
Stopped  ServiceLayer       ServiceLayer
Running  wscsvc             Security Center

PS> Get-Service -DisplayName ServiceLayer,Server

Status   Name               DisplayName
------   ----               -----------
Running  lanmanserver       Server
Stopped  ServiceLayer       ServiceLayer

<span data-ttu-id="3dfd7-112">Параметр ComputerName командлета Get-Service можно использовать для получения служб на удаленных компьютерах.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-112">You can use the ComputerName parameter of the Get-Service cmdlet to get the services on remote computers.</span></span> <span data-ttu-id="3dfd7-113">Параметр ComputerName принимает несколько значений и подстановочные знаки, что позволяет получить службы на нескольких компьютерах с помощью одной команды.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-113">The ComputerName parameter accepts multiple values and wildcard characters, so you can get the services on multiple computers with a single command.</span></span> <span data-ttu-id="3dfd7-114">Например, приведенная ниже команда получает службы на удаленном компьютере Server01.</span><span class="sxs-lookup"><span data-stu-id="3dfd7-114">For example, the following command gets the services on the Server01 remote computer.</span></span>
