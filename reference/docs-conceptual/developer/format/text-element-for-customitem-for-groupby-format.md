# <a name="text-element-for-customitem-for-groupby-format"></a>Элемент Text для элемента CustomItem для элемента GroupBy (формат)

Задает текст, который добавляется в данные, отображаемые элементом управления, например метку, скобки для заключения данных, а также пробелы для отступа данных. Этот элемент используется при определении того, как отображается новая группа объектов.

Элемент конфигурации (Format) Виевдефинитионс элемент (Format) элемент представления (формат) элемент GroupBy для элемента Ошибка customcontrol представления (Format) для элемента Кустоментриес (Format) для ошибка customcontrol для элемента GroupBy (Format) Кустоментри для Ошибка customcontrol для элемента Кустомитем GroupBy (Format) для Кустоментри для элемента GroupBy (Format) Text для Кустомитем для GroupBy (Format)

## <a name="syntax"></a>Синтаксис

```xml
<Text>TextToDisplay</Text>
```

В следующих разделах описываются атрибуты, дочерние элементы и родительский элемент элемента `Text`.

### <a name="attributes"></a>Атрибуты

Нет.

Нет.

|Элемент|Описание|

## <a name="text-value"></a>Текстовое значение

Укажите текст элемента управления для данных, которые требуется отобразить.
