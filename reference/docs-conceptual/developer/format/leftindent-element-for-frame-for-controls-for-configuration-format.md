# <a name="leftindent-element-for-frame-for-controls-for-configuration-format"></a>Элемент LeftIndent для элемента Frame для элемента Controls для элемента Configuration (формат)

Указывает, сколько символов перемещает данные из левого поля. Этот элемент используется при определении общего элемента управления, который может использоваться всеми представлениями в файле форматирования.
