# <a name="wraptables-element-format"></a>Элемент WrapTables (формат)

Указывает, что данные в ячейке таблицы перемещаются на следующую строку, если данные длиннее, чем ширина столбца.

Элемент Configuration (Format) Дефаултсеттингс элемент (Format) Враптаблес (формат)

## <a name="syntax"></a>Синтаксис

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние элементы и родительский элемент элемента `WrapTables`.

### <a name="attributes"></a>Атрибуты

Нет.

### <a name="child-elements"></a>Дочерние элементы

Нет.
