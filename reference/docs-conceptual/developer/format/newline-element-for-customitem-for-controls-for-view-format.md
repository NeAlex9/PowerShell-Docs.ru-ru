# <a name="newline-element-for-customitem-for-controls-for-view-format"></a>Элемент NewLine для элемента CustomItem для элемента Controls для элемента View (формат)

Добавляет пустую строку к отображению элемента управления. Этот элемент используется при определении элементов управления, которые могут использоваться представлением.

## <a name="syntax"></a>Синтаксис

```xml
<NewLine/>
```
