# <a name="rightindent-element-for-frame-for-groupby-format"></a>Элемент RightIndent для элемента Frame для элемента GroupBy (формат)

Указывает, сколько символов перемещает данные с правого края. Этот элемент используется при определении того, как отображается новая группа объектов.

Элемент конфигурации (Format) Виевдефинитионс элемент (формат) элемент GroupBy для элемента Ошибка customcontrol представления (Format) для элемента Кустоментриес для GroupBy (Format) Кустоментри для ошибка customcontrol для ошибка customcontrol for GroupBy (формат) кустомитем элемент for кустоментри для элемента в кустомитем for GroupBy (Format) ригхтиндент для элемента for GroupBy (формат)

## <a name="syntax"></a>Синтаксис

```xml
<RightIndent>CharactersToShift</RightIndent>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние элементы и родительский элемент `RightIndent` элемента.

Отсутствует.

### <a name="child-elements"></a>Дочерние элементы

Отсутствует.

## <a name="text-value"></a>Текстовое значение
