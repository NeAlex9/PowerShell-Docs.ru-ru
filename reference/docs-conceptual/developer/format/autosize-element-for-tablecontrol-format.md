# <a name="autosize-element-for-tablecontrol-format"></a>Элемент AutoSize для TableControl (формат)

Указывает, изменяется ли размер столбца и число столбцов в зависимости от размера данных.

Элемент Виевдефинитионс (Format) представления (Format) Таблеконтрол элемент (Format) AutoSize для Таблеконтрол (Format)

## <a name="syntax"></a>Синтаксис

```xml
<AutoSize/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние элементы и родительский элемент элемента `AutoSize`.

### <a name="attributes"></a>Атрибуты

Нет.

### <a name="child-elements"></a>Дочерние элементы

Нет.
