# <a name="format-schema-xml-reference"></a>Ссылка на схему формата XML

В подразделах этого раздела описываются XML-элементы, используемые файлами форматирования (Format. ps1xml Files). Файлы форматирования определяют, как будет отображаться объект .NET. они не изменяют сам объект.
