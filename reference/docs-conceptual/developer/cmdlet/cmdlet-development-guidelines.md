# <a name="cmdlet-development-guidelines"></a>Рекомендации по разработке командлетов

В подразделах этого раздела приводятся рекомендации по разработке, которые можно использовать для создания командлетов правильного формата. Используя общие функциональные возможности среды выполнения Windows PowerShell и, следуя этим рекомендациям, можно разрабатывать надежные командлеты с минимальными усилиями и обеспечить единообразную работу пользователя. Кроме того, вы уменьшите нагрузку на тест, так как для них не требуется повторное тестирование.
