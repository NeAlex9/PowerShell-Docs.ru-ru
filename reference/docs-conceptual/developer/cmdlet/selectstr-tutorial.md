# <a name="selectstr-tutorial"></a>Руководство по SelectStr

В этом разделе содержится учебник по созданию командлета Select-str, который очень похож на командлет [Select-String](/powershell/module/microsoft.powershell.utility/select-string) , предоставляемый Windows PowerShell. В этом учебнике содержатся фрагменты кода, иллюстрирующие реализацию командлетов и объяснение кода.
