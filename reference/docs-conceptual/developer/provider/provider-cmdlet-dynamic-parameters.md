# <a name="provider-cmdlet-dynamic-parameters"></a><span data-ttu-id="ad253-102">Динамические параметры командлета поставщика</span><span class="sxs-lookup"><span data-stu-id="ad253-102">Provider cmdlet dynamic parameters</span></span>

<span data-ttu-id="ad253-103">Поставщики могут определять динамические параметры, добавляемые в командлет поставщика, когда пользователь указывает определенное значение для одного из статических параметров командлета.</span><span class="sxs-lookup"><span data-stu-id="ad253-103">Providers can define dynamic parameters that are added to a provider cmdlet when the user specifies a certain value for one of the static parameters of the cmdlet.</span></span> <span data-ttu-id="ad253-104">Например, поставщик может добавить различные динамические параметры в зависимости от того, какой путь пользователь указывает при вызове командлетов поставщика `Get-Item` или `Set-Item`.</span><span class="sxs-lookup"><span data-stu-id="ad253-104">For example, a provider can add different dynamic parameters based on what path the user specifies when they call the `Get-Item` or `Set-Item` provider cmdlets.</span></span>

Management. автоматизации. Provider. итемкмдлетпровидер. жетитемдинамикпараметерс \*](/dotnet/api/System.Management.Automation.Provider.ItemCmdletProvider.GetItemDynamicParameters) , взятого из поставщика сертификатов:</span><span class="sxs-lookup"><span data-stu-id="ad253-108">Here is an example of an implementation of the [System.Management.Automation.Provider.Itemcmdletprovider.Getitemdynamicparameters\*](/dotnet/api/System.Management.Automation.Provider.ItemCmdletProvider.GetItemDynamicParameters) method taken from the Certificate provider:</span></span>

```csharp

    return new CertificateProviderDynamicParameters();
```

<span data-ttu-id="ad253-109">В отличие от статических параметров командлетов поставщика, можно указать характеристики этих параметров таким же образом, как параметры определяются в отдельных командлетах.</span><span class="sxs-lookup"><span data-stu-id="ad253-109">Unlike the static parameters of provider cmdlets, you can specify the characteristics of these parameters in the same way that parameters are defined in stand-alone cmdlets.</span></span> <span data-ttu-id="ad253-110">Ниже приведен пример класса динамических параметров, полученного от поставщика сертификатов.</span><span class="sxs-lookup"><span data-stu-id="ad253-110">Here is an example of a dynamic parameter class taken from the Certificate provider:</span></span>

```csharp
internal sealed class CertificateProviderDynamicParameters
{
  /// <summary>
  /// Dynamic parameter the controls whether we only return
  /// code signing certs.
  /// </summary>
  [Parameter()]
  public SwitchParameter CodeSigningCert
  {
    get
    {
      {
        return codeSigningCert;
      }
    }

    set
    {
      {
        codeSigningCert = value;
      }
    }
  }

    private SwitchParameter codeSigningCert = new SwitchParameter();
}
```
