# <a name="getproc02-code-samples"></a>Примеры кода GetProc02

Ниже приведены примеры кода для командлета GetProc02 Sample. Это пример командлета `Get-Process`, описанный в разделе [Добавление параметров, обрабатывающих входные данные командной строки](../cmdlet/adding-parameters-that-process-command-line-input.md). Этот командлет `Get-Process` извлекает процессы на основе их имени, а затем отображает сведения о процессах в командной строке.

> [!NOTE]
> Вы можете скачать C# исходный файл (getproc02.cs) для этого командлета Get-proc с помощью пакета средств разработки программного обеспечения Microsoft Windows для компонентов среды выполнения Windows Vista и .NET Framework 3,0. Инструкции по загрузке см. в статье [Установка Windows PowerShell и Загрузка пакета SDK для Windows PowerShell](/powershell/scripting/developer/installing-the-windows-powershell-sdk).
>
> Скачанные исходные файлы доступны в **\<примеров PowerShell >** Directory.
