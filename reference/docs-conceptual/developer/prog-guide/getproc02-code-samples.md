# <a name="getproc02-code-samples"></a>Примеры кода GetProc02

Ниже приведены примеры кода для командлета GetProc02 Sample. Это `Get-Process` Пример командлета, описанный в разделе [Добавление параметров, обрабатывающих входные данные командной строки](../cmdlet/adding-parameters-that-process-command-line-input.md). Этот `Get-Process` командлет извлекает процессы на основе их имени, а затем отображает сведения о процессах в командной строке.

> [!NOTE]
> Исходный файл C# (getproc02.cs) для этого командлета Get-proc можно скачать с помощью пакета средств разработки программного обеспечения Microsoft Windows для компонентов среды выполнения Windows Vista и .NET Framework 3,0. Инструкции по загрузке см. в статье [Установка Windows PowerShell и Загрузка пакета SDK для Windows PowerShell](/powershell/scripting/developer/installing-the-windows-powershell-sdk).
>
> Скачанные исходные файлы доступны в **\<PowerShell Samples>** каталоге.
