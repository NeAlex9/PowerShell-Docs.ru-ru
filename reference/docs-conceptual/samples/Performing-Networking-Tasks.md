<span data-ttu-id="f9f0c-121">Удобнее отображать сводные данные, содержащие свойства Address, ResponseTime и StatusCode, как это делает приведенная ниже команда.</span><span class="sxs-lookup"><span data-stu-id="f9f0c-121">A more useful form for summary information a display of the Address, ResponseTime, and StatusCode properties, as generated by the following command.</span></span> <span data-ttu-id="f9f0c-122">Параметр Autosize командлета Format-Table изменяет размер столбцов таблицы для их правильного отображения в Windows PowerShell.</span><span class="sxs-lookup"><span data-stu-id="f9f0c-122">The Autosize parameter of Format-Table resizes the table columns so that they display properly in Windows PowerShell.</span></span>

### <a name="releasing-and-renewing-dhcp-leases-on-specific-adapters"></a><span data-ttu-id="f9f0c-151">Отмена и обновление аренды адреса DHCP для отдельных адаптеров</span><span class="sxs-lookup"><span data-stu-id="f9f0c-151">Releasing and Renewing DHCP Leases on Specific Adapters</span></span>
